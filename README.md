# Hydroponics Autodoser + IoT 
Read pH, EC, control pumps, send data to Thingspeak.

Goal: Automate nutrient control in a hydroponics system.

## Modify parameters in parameters.h
```
// upload_data
String apiKey = "xxx";

// pump_activation

float phLowerThreshold = 5.8;
```
